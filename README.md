# User Service API

這是一個使用 Golang 和 Gin 框架實現的簡單 RESTful API。
swagger 文檔: http://localhost:8080/swagger/index.html#/

## 功能特點

- RESTful API 設計

## API 端點說明

### 1. 創建用戶
- **路徑**: POST /api/v1/users
- **描述**: 創建新用戶
- **請求體**:
```json
{
"username": "string",
"email": "string",
"password": "string"
}
```
